About This Demo

This page streams AI model output through a Braille-native codec. The Braille-Encoded section contains Unicode Braille characters (U+2800–U+28FF) that your refreshable Braille display will render as physical dot patterns.

The Confidence Track shows one Braille cell per token where dot density represents model certainty: ⣿ = very confident, ⠀ = very uncertain. If you feel the cells becoming sparser, the model is less sure of what it's saying.

For researchers: This is part of the Synapse.js Braille-native AI project. We're looking for Braille display users to validate the tactile experience. Contact us if interested.
